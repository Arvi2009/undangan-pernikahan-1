⚡ Optimasi Sebelum Deploy

1. Compress Gambar

Kurangi ukuran file gambar tanpa mengurangi kualitas:

  • TinyPNG - Compress PNG & JPG
  • ImageCompressor - Online tool
  • Tools lokal: ImageMagick, FFmpeg

2. Minify CSS & JavaScript

// Gunakan online tools atau build tools
// Online: minifier.org, jscompress.com
// Tools: npm, webpack, gulp

// Atau gunakan:
  • JavaScript Minifier
  • CSS Minifier

3. Lazy Load Images

<img src="placeholder.jpg" data-src="real-image.jpg" loading="lazy" />

4. Enable Gzip Compression

Aktifkan di server settings untuk mengurangi ukuran transfer

5. Caching Strategy

# Buat .htaccess untuk caching
<IfModule mod_expires.c>
  ExpiresActive On
  ExpiresByType text/html "access plus 1 week"
  ExpiresByType image/jpeg "access plus 1 year"
</IfModule>

📦 Deployment ke Netlify (Cara Termudah)

🎯 Opsi 1: Drag & Drop (Paling Mudah)

1. Buka https://app.netlify.com
2. Sign up dengan email atau GitHub
3. Buat folder dengan semua file (index.html, css/, js/)
4. Drag & drop folder ke Netlify
5. Tunggu selesai, klik link yang muncul
6. (Opsional) Beli domain atau gunakan subdomain gratis

⚙️ Opsi 2: Connect GitHub (Lebih Powerful)

1. Push kode ke GitHub repository
2. Di Netlify, pilih "Connect to Git"
3. Pilih GitHub repository Anda
4. Set build settings (skip untuk static site)
5. Deploy! Auto-deploy saat ada push ke main branch

🔗 Setup Custom Domain

🌍 Menggunakan Domain Sendiri

1. Beli domain di Namecheap, GoDaddy, atau hosting provider lain
2. Di Netlify: Site Settings → Domain Management → Add Custom Domain
3. Follow instruksi untuk update DNS nameserver
4. DNS propagasi butuh 24-48 jam
5. Verifikasi selesai, HTTPS auto-setup

❓ FAQ

Q: Apakah website akan crash jika banyak visitor?

A: Untuk static site di Netlify/Vercel, tidak akan crash. CDN mereka bisa handle jutaan request. RSVP form hanya menyimpan di localStorage lokal, bukan server.

Q: Bagaimana jika ingin save RSVP ke database?

A: Gunakan backend service gratis seperti Firebase, Supabase, atau Formspree untuk handle form submission.

Q: Bisa pakai domain gratis selamanya?

A: Ya! Netlify/Vercel kasih subdomain gratis selamanya. Atau pakai namecheap yang murah (~Rp10-20rb/tahun).

Q: Bagaimana share link ke 500+ tamu?

A: Gunakan Excel/Google Sheets + script untuk generate link, atau kirim via WhatsApp bulk message.

Q: Apakah data guestbook disimpan permanent?

A: Data tersimpan di localStorage browser tamu. Untuk permanent storage, gunakan backend/database.
