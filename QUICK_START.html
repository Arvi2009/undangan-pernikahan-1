Quick Start Guide

✓ Website siap digunakan! Ikuti panduan ini untuk personalisasi.

📋 Langkah 1: Memahami Struktur File

1

File-file Anda

Pastikan Anda memiliki struktur folder seperti ini:

📁 UNDANGAN PERNIKAHAN/
├── 📄 index.html
├── 📁 css/
│ └── 📄 style.css
├── 📁 js/
│ ├── 📄 config.js
│ └── 📄 main.js
├── 📄 README.md
└── 📄 QUICK_START.html (file ini)

✏️ Langkah 2: Edit Informasi Pengantin

2

Buka js/config.js dan ubah:

const weddingConfig = {
    brideGroom: {
        groom: 'NAMA_PRIA_ANDA',      // ← Ganti
        bride: 'NAMA_WANITA_ANDA',    // ← Ganti
        shortNames: 'GANTI & GANTI'   // ← Ganti
    },
    eventDate: {
        date: '2024-05-25',           // ← Ganti (YYYY-MM-DD)
        year: 2024,
        month: 5,
        day: 25
    }
}

🏢 Langkah 3: Update Lokasi Acara

3

Di file js/config.js, ubah lokasi:

events: {
    akad: {
        time: '09:00',
        endTime: '10:00',
        location: 'NAMA_MASJID_ANDA'  // ← Ganti
    },
    resepsi: {
        time: '11:00',
        endTime: '16:00',
        location: 'NAMA_GEDUNG_ANDA'  // ← Ganti
    }
}

📱 Langkah 4: Update Kontak WhatsApp

4

Di file js/config.js, ubah nomor WhatsApp:

contacts: {
    whatsapp: '62812345678',  // ← Ganti (format: 62...)
    instagram: 'username_anda',
    email: 'email@anda.com'
}

Format WhatsApp: Gunakan format 62, contoh: 62812345678

🎶 Langkah 5: Update Musik Background (Opsional)

5

Buka index.html dan cari elemen <audio>:

<audio id="bgMusic" loop>
    <source src="GANTI_URL_MUSIK.mp3" type="audio/mpeg">
</audio>

Sumber Musik Gratis:

  • Pixabay Music (https://pixabay.com/music/)
  • FreePik (https://freepik.com/)
  • YouTube Audio Library

📸 Langkah 6: Tambah Galeri Foto

6

Edit section #galeri di index.html:

<a href="LINK_FOTO_BESAR.jpg" data-lightbox="gallery">
    <img src="LINK_FOTO_KECIL.jpg" alt="Deskripsi">
    <div class="gallery-overlay">
        <i class="fas fa-search-plus"></i>
    </div>
</a>

Opsi Foto:

  • Gunakan URL dari Unsplash/Pixabay (gratis)
  • Upload ke ImgBB, Imgur, atau layanan hosting foto
  • Gunakan foto lokal jika di-host sendiri

💕 Langkah 7: Update Love Story

7

Edit section #cerita untuk cerita pribadi Anda

<div class="story-item">
    <div class="story-content">
        <h3 class="story-year">2020</h3>
        <h4 class="story-title">Judul Cerita</h4>
        <p class="story-text">Cerita indah kalian...</p>
    </div>
</div>

🗺️ Langkah 8: Update Google Maps (Opsional)

8

Untuk mengubah lokasi maps:

  1. Buka Google Maps
  2. Cari lokasi acara Anda
  3. Klik "Share" → "Embed a map"
  4. Copy iframe code
  5. Paste di section #acara di index.html

🎥 Langkah 9: Update Video Prewedding

9

Ubah video YouTube di section #galeri:

<iframe 
    src="https://www.youtube.com/embed/VIDEO_ID"
    title="Prewedding Video" 
    frameborder="0">
</iframe>

Cara mendapat VIDEO_ID:

  • Buka video di YouTube
  • URL: https://www.youtube.com/watch?v=dQw4w9WgXcQ
  • Bagian bold adalah VIDEO_ID

🎨 Langkah 10: Ganti Tema Warna (Opsional)

10

Edit css/style.css di bagian atas:

:root {
    --primary-gold: #D4A574;    /* Warna emas - ganti di sini */
    --dark-brown: #3D2817;      /* Warna teks - ganti di sini */
    --light-pink: #FFF0F5;      /* Warna pink - ganti di sini */
}

Kombinasi Warna Rekomendasi:

Tema Gold Brown
Classic Gold #D4A574 #3D2817
Rose Gold #E8A5C1 #6B3E42
Silver Elegance #C0C0C0 #2C2C2C

🔗 Langkah 11: Buat Invitation Link Unik

11

Untuk setiap tamu, gunakan format:

https://yoursite.com/?to=NamaTamu

Contoh:

  • https://ayureza.com/?to=Bambang%20Irawan
  • https://ayureza.com/?to=Siti%20Nurhaliza
  • https://ayureza.com/?to=Ahmad%20Pratama

Tips: Ganti spasi dengan %20, atau gunakan URL encoder online

🚀 Langkah 12: Test & Deploy

12

Sebelum deploy:

  • ✓ Buka index.html di browser (Chrome, Firefox, Safari)
  • ✓ Test di desktop, tablet, dan mobile
  • ✓ Test countdown timer
  • ✓ Test musik play/pause
  • ✓ Test RSVP form
  • ✓ Test galeri foto (lightbox)
  • ✓ Test toggle bahasa

Deploy ke Hosting Gratis:

  • Netlify: Drag & drop folder, instant live
  • Vercel: Connect GitHub, auto-deploy
  • GitHub Pages: Push ke GitHub, enable Pages
  • Firebase: Firebase CLI commands
⚠️ PENTING:
  • Pastikan semua file dalam satu folder
  • Tidak perlu mengubah CSS atau JavaScript jika hanya personalisasi
  • Ganti URL foto & musik dengan yang valid
  • Test semua fitur sebelum membagikan link
💡 TIPS TAMBAHAN:
  • Gunakan foto berkualitas tinggi (minimum 1200x1200px)
  • Compress foto untuk mempercepat loading: https://tinypng.com/
  • Test di berbagai browser
  • Update konten berdasarkan feedback tamu
  • Share link unik ke setiap tamu untuk tracking personal
