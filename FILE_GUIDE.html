🎯 Panduan Memulai (Step by Step)

1

Baca QUICK_START.html

Mulai dari panduan cepat untuk setup awal dalam 12 langkah mudah

2

Edit js/config.js

Ganti nama pengantin, tanggal, lokasi, dan kontak WhatsApp

3

Update index.html

Tambah galeri foto, update love story, ganti musik background

4

Customize Design (Opsional)

Edit warna di css/style.css, ganti font, atau baca CUSTOMIZATION_EXAMPLES.js

5

Test Locally

Buka index.html di browser dan test semua fitur (desktop & mobile)

6

Baca DEPLOYMENT_GUIDE.html

Ikuti panduan deploy ke hosting (Netlify/Vercel/GitHub Pages)

7

Deploy & Go Live

Push ke hosting, setup domain, dan bagikan link ke tamu

🔖 Quick Reference

Untuk Mengubah Konten:
  • Nama pengantin → Edit config.js & index.html
  • Tanggal & waktu → Edit config.js & index.html
  • Teks bahasa → Edit config.js (languages object)
  • Warna tema → Edit style.css (:root variables)
  • Musik → Edit index.html (audio src)
  • Galeri foto → Edit index.html (gallery section)
Untuk Menambah Fitur:
  • Custom animasi → Lihat CUSTOMIZATION_EXAMPLES.js
  • Social sharing → Copy dari CUSTOMIZATION_EXAMPLES.js
  • Analytics → Copy tracking function dari examples
  • Backend integration → Edit main.js form handler
